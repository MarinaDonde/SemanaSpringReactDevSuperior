# Como executar o frontEnd:

```bash
# Clonar repositório:
git clone https://github.com/MarinaDonde/SpringReact.git

# Entrar na pasta do projeto AgViaje
cd springreact

# Entrar na pasta do projeto frontend
cd frontend

# Instalar dependências
yarn install

# Executar o projeto
yarn start
```
